INFORMACION BASICA

Lenguajes de programacion:Son sistemas formales que permiten a los programadores comunicarse con las computadoras.
Algunos ejemplos populares son Python, Java, C++ y JavaScript. Sintaxis:La sintaxis se refiere a las reglas y estructuras
que debes seguir al escribir código en un lenguaje de programación. Es importante para que el programa funcione correctamente.
Variables: Las variables son contenedores para almacenar datos. Pueden ser números, texto o cualquier otro tipo de información.
Instrucciones condicionales:Estas permiten que el programa tome decisiones basadas en condiciones. Por ejemplo, “si X es verdadero, haz Y”.
Bucles: Los bucles permiten repetir una serie de instrucciones varias veces. El bucle "for" y el bucle “while” son comunes.
Funciones: Las funciones son bloques de código reutilizables que realizan una tarea específica. Ayudan a organizar y modular el código.
Algoritmos: Son secuencias de pasos para resolver un problema. La programación se trata de diseñar algoritmos eficientes.

LA PROGRAMACION

La programación es el acto de organizar una secuencia de pasos ordenados para lograr una tarea específica.
En el ámbito de la informática, se refiere a la creación de programas o aplicaciones mediante el desarrollo de un código fuente.
Este código, escrito en lenguajes de programación como Java, Pearl o Python, comunica al ordenador las instrucciones necesarias para ejecutar programas.
Así, la programación permite que los ordenadores funcionen y realicen tareas solicitadas por los usuarios.
